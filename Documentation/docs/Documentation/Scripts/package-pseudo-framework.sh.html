package-pseudo-framework.sh Document

    This script is used in iOS targets that are packaged up as “pseudo” frameworks.

    The script is called from a Run Script phase, like this:

    “${ECCONFIG_SCRIPTS_PATH}/package-pseudo-framework.sh”

    It performs various linking and copying operations to lay out the framework bundle correctly.

    prefix#!/usr/bin/env bash prefix prefix## This script is used in iOS targets that are packaged up as “pseudo” frameworks. prefix## prefix## The script is called from a Run Script phase, like this: prefix## prefix## “${ECCONFIG_SCRIPTS_PATH}/package-pseudo-framework.sh” prefix## prefix## It performs various linking and copying operations to lay out the framework bundle correctly. prefix prefixFRAMEWORK_ROOT=“${BUILT_PRODUCTS_DIR}/${CONTENTS_FOLDER_PATH}” prefix prefixmkdir -p “${FRAMEWORK_ROOT}/Versions” prefix/bin/ln -sfh A “${FRAMEWORK_ROOT}/Versions/Current” prefix/bin/ln -sfh Versions/Current/Headers “${FRAMEWORK_ROOT}/Headers” prefix/bin/ln -sfh Versions/Current/Resources “${FRAMEWORK_ROOT}/Resources” prefix/bin/ln -sfh “Versions/Current/${PRODUCT_NAME}” “${FRAMEWORK_ROOT}/${PRODUCT_NAME}”
